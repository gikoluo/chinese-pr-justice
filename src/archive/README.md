- `RemovedFromOTT313.astro` preserves the VAN501-derived event details, schedule, guidelines, and CTA section that was removed from `src/pages/events/OTT313-2026.astro`. It is kept only as maintenance history and is not imported by the live OTT313 page.  -- By DEZHI WANG
